Recent projects

Birthday Newz

Birthday Newz takes a date in recent history, and digs up enough information from the web to build a newspaper around that date. The user is able to write an article, heading and upload a picture - and then print or post to Facebook.

It uses AngularJS on the front end - perfect for data binding between the user and the newspaper, and Rails on the backend.

Birthday Newz does some real time scraping - getting information from other web sites only when the user enters a date. Some other information is pre-scraped, and stored in a Postgres database.

fritter

Inspired by Pinterest, fritter is a mood board for collecting favourite snippets. A user may sign up, log in, and collect 'favourites' to show on their own page.

Written with Rails on the back end, and AngularJS on the front, fritter uses a graphics package called 'Masonry' to address the problem of laying out cells of varying height.

wantMAW

Written with Ruby on Rails - front and back end, wantMAW connects translators with those who have project based work available.

I used nested forms to access data through it's associated data - eg accessing the 'language profiency' table through the user form.
